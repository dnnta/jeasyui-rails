jquery-easyui-rails
===================

## INSTALLATION

You just have to add the 'jeasyui-rails' gem to your Gemfile

#### Gemfile

    gem 'jeasyui-rails'

#### application.js 

    //= require jeasyui-rails

if need locale

    //= require jeasyui/locale/jp

#### application.css

    *= require jeasyui-rails

if need theme

    *= require jeasyui-rails/themes/¥[theme_name_here¥]

enjoy!
